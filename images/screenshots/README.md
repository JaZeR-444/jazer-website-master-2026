# Screenshots Directory

This directory contains visual previews of the JaZeR website for the README.

## Required Screenshots

### 1. `desktop-preview.png`
**Description**: Full homepage view on desktop (1920x1080)
**What to capture**:
- Hero section with 600x200px bubble logo
- Gradient background
- Navigation bar with logo
- Main call-to-action buttons
- Sample content cards

**How to capture**:
1. Open https://jazer-music.com in browser
2. Set viewport to 1920x1080 (full screen)
3. Take screenshot (Win + Shift + S or Cmd + Shift + 4)
4. Save as `desktop-preview.png`

---

### 2. `mobile-preview.png`
**Description**: Mobile view showing responsive design (375x812 - iPhone X)
**What to capture**:
- Mobile navigation with hamburger menu
- Responsive layout
- Touch-friendly buttons
- Stacked content cards

**How to capture**:
1. Open https://jazer-music.com in browser
2. Open DevTools (F12) → Toggle device toolbar (Ctrl/Cmd + Shift + M)
3. Select "iPhone X" or set custom 375x812
4. Take screenshot from DevTools
5. Save as `mobile-preview.png`

---

### 3. `pages-overview.png`
**Description**: Grid/collage showing all main pages
**What to capture**:
- Home page thumbnail
- Music page thumbnail
- Videos page thumbnail
- About page thumbnail
- Shop page thumbnail
- Contact page thumbnail

**How to capture**:
1. Take individual screenshots of each page (at ~800px wide)
2. Use image editor (Photoshop, Figma, Canva) to create grid layout
3. Arrange in 2x3 or 3x2 grid
4. Add subtle borders/shadows for separation
5. Save as `pages-overview.png`

---

## Image Specifications

- **Format**: PNG (for transparency and quality)
- **Max width**: 800px (desktop), 400px (mobile)
- **Optimization**: Use TinyPNG or similar to compress
- **Alt text**: Must be descriptive for accessibility
- **File size**: Keep under 500KB per image

## Tools for Screenshots

- **Browser DevTools**: Built-in responsive design mode
- **Windows**: Snipping Tool (Win + Shift + S)
- **macOS**: Screenshot utility (Cmd + Shift + 4)
- **Extensions**: 
  - GoFullPage (full-page screenshots)
  - Awesome Screenshot
  - Nimbus Screenshot

## Tools for Editing/Compositing

- **Figma**: Free, web-based, great for layouts
- **Canva**: Easy grid creation
- **GIMP**: Free Photoshop alternative
- **Photoshop**: Professional option
- **Preview (macOS)**: Basic editing
